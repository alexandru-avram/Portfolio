# Python Projects

### __beginner
Just some basic Jupyter notebooks, showcasing beginner exercises, working with NummPy and Pandas data analytics exercises, ML exercises and some games.

### 01. Text to Morse Code Converter
Simple app where where I convert a text into a morse code message. No sounds, don't worry.

### 02. Snake Game
Classic "Snake" done using tkinter and turtle. Has a highscore.

### 03. Color Extractor Website
It's a flask run HTML project where you can upload a photo and get the top 10 most used colors.

### 04. Steam Top 50 Games
A scrapper that gets the top 50 games on the Steam Charts and pulls up some statistics about them.

### 05. Space Race Analysis
Data exploration notebook about the Space Race and space exploration.

### 06. Cafes in London
Flask project with SQL Lite db API based on an exercise from the Udemy course "100 days of Python"
